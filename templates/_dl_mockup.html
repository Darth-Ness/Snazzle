Download for Windows

Instructions

Installer (.msi)

  1. Download the installer
  2. Run it with your favourite settings
  3. You can now launch Snazzle from wherever

Portable (.zip)

  1. Download the file
  2. Unzip on your favourite USB drive
  3. You can now take Snazzle with you, wherever, or install it without admin priveliges

Download for macOS

Instructions

Installer (.dmg)

  1. Download the installer
  2. Double-click it to attach it
  3. Drag Snazzle to your Applications folder
  4. You can now launch Snazzle from wherever
